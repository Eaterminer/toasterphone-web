Toasterphone

A Discord bot for playing the classic drawing and guessing game Toaster Phone!

Commands:

  • !web - Opens the game website
  • !settings - Adjust game settings
  • !join - Join the game
  • !host - Host a new game
  • !leave - Exit the game

Info:

Welcome to Toasterphone! This bot allows you to play a fun drawing and guessing game with your friends in Discord. Join the game, guess the drawing, or host your own game and let others guess your drawing!
